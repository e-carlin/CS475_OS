#Reverse Polish Notation Calculator

Evan Carlin
2/12/2017

A program to evaluate equations in Reverse
Polish Notation. The calculator supports
+, -, *, and / operations. Enter operators
and operands separated by space, newline, or
tab characters.

To make:
	$ make all

To run:
	$ ./calculator

To clean:
	$ make clean

Note:
This was compiled on the provided 32-Bit Ubuntu  
image using gcc.
